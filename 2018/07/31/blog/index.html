部署时候遇到的坑

在发布和部署的时候主要会遇到两类坑。
一类就是缺少依赖，这种问题解决办法就是按照缺少的依赖一步一步添加。
一类就是会出现无法访问的问题：
本地服务不能启动，或者是启动时没有报错但是访问不了，删除 node_modules ，重新 yarn install 安装。
部署之后 github 上面的文件是一些奇怪的字符。将 github 上的项目删除，重新新建项目，重新部署。
部署之后 github 上的文件是正常的，但是不能访问。尝试重新部署一次看是否可以解决。
